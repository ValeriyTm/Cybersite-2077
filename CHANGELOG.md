## - 2026-04-14 (v. 1.10.2)

### Added

- К проекту подключены Nginx, Prometheus и Grafana Loki + Winston.

### Changed

- Произведен полный рефакторинг бэкенд-кода. Всё приведено к стилю MVC, улучшена обработка ошибок и деление кода на модули.

### Added

## - 2026-04-11 (v. 1.10.0)

### Added

- Реализован фунционал модуля Content - теперь пользователи с ролью CONTENT_EDITOR, AMDIN и SUPERADMIN могут использовать кастомный конструктор для создания страницы новостей.
- Реализован функционал модуля Admin - создана кастомная админ-панель. Админ-панель работает по модели RBAC и позволяет работать с каталогом (CRUD-операции), с заказами, тикетами поддержки, промоакциями и скидками, отчетами, управлением доступом и создавать контент для страницы новостей.

## - 2026-04-04 (v. 1.3.0)

### Added

- Реализован функционал складов и заказов (модули Warehouse и Ordering).
- Добавлены страницы создания заказа и страница заказов пользователя.
- Добавлены в БД PostgreSQL таблицы складов, остатков на складах, заказов, товаров в заказах. Данные по остаткам были сгенерированы рандомно.
- Подключена географическая карта через leaflet.js для выбора адреса доставки и для отображения складов интернет-магазина. На сервере используется PostGIS.
- Каталог дополнен фильтром по наличию и информацией о наличии товара.
- Для реализации жизненного цикла заказа внедрен BullMQ. В процессе жизненного цикла изменяется статус заказа и остатки на складах.
- Реализован расчёт даты и стоимости доставки.

### Fixed

- Исправлена работа состояния счетчика количества товаров в избранном и корзине в Header.
